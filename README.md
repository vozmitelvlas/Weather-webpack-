# Цель проекта

Реализация интерактивного приложения «Погода». Проект демонстрирует:

- Применение современных подходов к сборке проекта с помощью Webpack
- Разработку на чистом JavaScript с разделением логики и представления
- Управление аудио, статическими ресурсами и пользовательским интерфейсом

# Решаемая задача

Создание веб-приложения, имитирующего смену погоды через визуальные и звуковые эффекты. Пользователь может выбрать тип погоды (солнце, дождь, снег), при этом появится соответствующий фон и атмосферный звук, громкость которого можно регулировать.

# Ключевые требования

- Поддержка трёх типов погоды: солнечно, дождь, снег
- Аудио не должны накладываться друг на друга, предыдущий звук должен выключаться при включении следующего
- При повторном нажатии на одну и ту же кнопку воспроизведение приостанавливается, а если после этого нажать еще раз, то воспроизведение продолжится
- После нажатия на кнопку для страницы устанавливается соответствующая фоновая картинка

# Запуск проекта

### Требования

- node.js версия 16+
- npm или yarn

### Инструкция по устрановке

1. Клонируйте репозиторий:
   `git clone https://github.com/vozmitelvlas/Weather-webpack-`
2. Установите зависимости
   `npm install #или yarn install`

3) Запустите проект в режиме разработки:
   `npm run dev # или yarn dev`
4) Откройте в браузере
   [http://localhost:3000/](http://localhost:3000/)

### Сборка для продакшена

`npm run build #или yarn build`

Собранное приложение будет находиться в папке dist/ и готово к размещению на любом статическом хостинге
